Programa para calcular la fracción irreducible de un número introducido por el usuario.
El programa está implementado en Python3, se adjuinta archivo para abrir con Jupyter. Se adjunta así mismo en formato ".py" para abrir desde cualquier otro editor.
En el código, se comentan los diferentes pasos que realiza el programa entre los cuáles:
   - Se comprueba si el número tiene 4 decimales
   - Se comprueba que el número está introducido entre el rango indicado.
   - Se verifica que el número decimal introducido usa el punto como separador decimal.
   - Se calcula la fracción irreducible.
